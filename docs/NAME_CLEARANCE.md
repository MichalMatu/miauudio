# Miauudio name screening

Screened: 2026-07-11.

This is a preliminary collision search, not a legal trademark opinion or a
guarantee that a store or registry will accept the name.

## Findings

- Exact quoted web search for `Miauudio`: no indexed result found.
- Exact Google Play app search: `No results for Miauudio`.
- Direct Google Play URL for `io.github.michalmatu.miauudio`: HTTP 404, so no
  public listing currently uses that package ID.
- Exact Apple App Store indexed search: no result found.
- Exact EUIPO-indexed web search: no result found.
- `miauudio.com`, `miauudio.app`, and `miauudio.pl` returned no A records during
  the check. WHOIS for `miauudio.com` returned `No match`.
- The existing `MichalMatu/miauudio` GitHub repository belongs to this project.

## Interpretation

No obvious exact-name collision was found, so `Miauudio` remains a reasonable
working store name. App titles on Google Play are not guaranteed to be unique;
the package ID is the technical unique identifier.

## Before a public launch

1. Search `Miauudio` and confusingly similar spellings directly in EUIPO
   eSearch/TMview and WIPO Global Brand Database for the intended software and
   audio-related classes.
2. Search Google Play and the Apple App Store again from the target countries.
3. Confirm the final developer/store name and countries of distribution.
4. If the project will be marketed commercially, obtain professional trademark
   advice before investing substantially in the brand.
5. Domain registration is a separate user decision; do not treat a DNS or
   WHOIS result as a reservation.
