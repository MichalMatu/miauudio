# Google Play graphics

- `play-icon.png`: 512 × 512 candidate generated from the currently approved
  logo without a pre-applied rounded mask.
- `feature-graphic.jpg`: 1024 × 500 candidate using the same current visual
  identity.
- The SVG files are editable sources for these rendered assets.

Phone screenshots must be captured from the signed release candidate. Prepare
four 1080 × 1920 portrait captures following `docs/GOOGLE_PLAY.md`; do not
substitute mockups or generated screens for the actual app UI.
